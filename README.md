# 3rd Person Shooter (Python)

A lightweight third-person style shooter built with **pygame**. The player moves with **WASD**, aims with the mouse, and shoots to survive waves of enemies.

## Requirements
- Python 3.9+
- pygame

## Install
```bash
pip install -r requirements.txt
```

## Run
```bash
python main.py
```

## Controls
- **WASD**: Move
- **Mouse**: Aim
- **Left Click**: Shoot
- **R**: Restart after game over
